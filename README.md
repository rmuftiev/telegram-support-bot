# Simple telegram bot for your Support department.

Bot code contains sections for FAQs and custom support messges. Custom messages are redirected to a private chat, where there Support specialist can reply them. Replies are then redirected back to the User.

## Instructions

1. Set your bot name and get your bot TOKEN_API via https://t.me/botfather

2. Add your bot in your telegram.

3. Create a private support channel for your support specialists and add your bot there.

4. Get the CHAT_ID number of your support channel. Usually, you can see it at the end of URL in telegram desktop, It should look something like '-1234567'.

5. Update the code with your TOKEN_API, CHAT_ID, and FAQ quesitons.

6. Deploy and run the script.

7. Enjoy.
